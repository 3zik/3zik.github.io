# High-Performance Monte Carlo Simulations of Ferrimagnetic Lattice Models

As part of my DOE SULI internship at Brookhaven National Laboratory (NSLS-II), I'm developing a high-performance C++ Monte Carlo simulation framework in the Condensed Matter Theory group, advised by Dr. Weiguo Yin. The project studies magnetic phase behavior in ferrimagnetic lattice models using atomistic spin-dynamics and Monte Carlo methods.

The framework implements Heat Bath and Replica Exchange (Parallel Tempering) algorithms to sample spin configurations efficiently across a range of temperatures, which helps navigate the complex energy landscapes that show up in ferrimagnetic systems. Simulations run on the [VAMPIRE](https://vampire.york.ac.uk/) atomistic spin-dynamics package on high-performance computing clusters.

Alongside the simulation work, I'm building automated pipelines for running and managing simulations at scale, and applying machine learning methods to analyze the resulting large simulation datasets.

This is ongoing work.
